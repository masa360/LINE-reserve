# Reservation Only 配布セット

このフォルダは **お客様配布用（予約-only）** の管理場所です。  
QA / 画像保存機能は含めません。

## 含めるもの（基準）

- GAS（正式名）: `gas/Code_reservation_only.gs`
- GAS（実体）: `gas/Code_reservation_only_base.gs`
- フロント: 予約導線のみ
- ドキュメント: 本番リリース手順（予約-only前提）

## 配布前チェック

1. 予約作成が成功する
2. カレンダーに正しく反映される
3. 重複予約が防止される
4. 予約ログが残る（必要時）
5. QA/画像保存の導線が出ていない

## 禁止事項

- もりもり版のコードをこの配布セットに混ぜること
